# E-Commerce Sales Analysis using Power BI

##  Project Overview
This project focuses on analyzing e-commerce sales data to identify trends, top-performing categories, and brand-level performance using Power BI.

##  Objectives
- Analyze monthly sales and order trends
- Identify high-demand product categories
- Find top-performing brands
- Build an interactive dashboard for business insights

##  Key Insights
- Highest sales observed in the month of **July**
- Maximum order volume recorded in **March**
- **Electronics and Fashion** are top-performing categories
- **Xiaomi** emerged as the best-performing brand
